Welcome to Priapus SlackBot

A slack bot that can be used to save slack channel conversation in Google drive
Slack App Link
Getting Started

Goto the Slack Workspace and register as a user..

Usage

To export conversation histroy of any channel there type:

/export-conversation

NOTE:
  • Ensure you have a Google account
  • If you are running this command the first time, you will be asked to allow Priapus SlackBot have write access to your Google Drive
